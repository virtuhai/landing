# Roadmap

## Funded six-month prototype programme

| Period | Deliverables |
| --- | --- |
| Months 1–2 | Architecture, security baseline, annual server environment, product requirements and data-source validation. |
| Months 2–4 | Data pipelines and adapters for priority APIs, map-based interface, alerting logic and internal prototype. |
| Months 5–6 | Integration and reliability testing, research with users and specialists, pilot-readiness documentation. |

After validation, VIRTUHAI will prepare pilot partnerships, strengthen operations and extend the platform only where the evidence supports it.

# Дорожня карта

## Фінансована шестимісячна програма прототипу

| Період | Результати |
| --- | --- |
| Місяці 1–2 | Архітектура, базові вимоги безпеки, серверне середовище на рік, вимоги до продукту та перевірка джерел даних. |
| Місяці 2–4 | Потоки даних і адаптери для пріоритетних API, інтерфейс з картою, логіка сповіщень та внутрішній прототип. |
| Місяці 5–6 | Інтеграційне тестування та перевірка надійності, дослідження з користувачами й спеціалістами, документація для готовності до пілоту. |

Після валідації VIRTUHAI готуватиме партнерства для пілотів, посилюватиме операційну частину й розширюватиме платформу лише там, де це підтверджено результатами.
