# Hardware

The initial VIRTUHAI programme is software-led. It focuses on a secure server environment, public-data integrations and a testable user experience before committing to custom hardware.

Future hardware research may evaluate field-ready sensors, wearable and vehicle form factors, connectivity, power use and maintenance. Hardware will be pursued only when it provides a validated advantage over trusted existing data sources and devices.

## Ukrainian version

# Обладнання

Початкова програма VIRTUHAI має програмний фокус. Вона зосереджена на захищеному серверному середовищі, інтеграціях публічних даних і досвіді користувача, який можна перевірити, до ухвалення рішення про власне обладнання.

Майбутні дослідження обладнання можуть оцінювати польові сенсори, носимі та автомобільні форм-фактори, зв’язок, енергоспоживання й обслуговування. Власне обладнання розвиватиметься лише там, де воно має перевірену перевагу над надійними наявними джерелами даних і пристроями.
