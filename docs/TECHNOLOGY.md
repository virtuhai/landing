# Technology

VIRTUHAI is planned as a modular software platform rather than a single monolithic application. Its core layers are:

1. **Source adapters** — controlled connectors to approved public APIs and datasets.
2. **Data pipeline** — normalisation, time and location context, provenance and quality checks.
3. **Awareness engine** — rules and research-driven models that rank signals and provide explanations.
4. **Product layer** — dashboard, alerts and future product-specific interfaces.
5. **Operations layer** — secure hosting, monitoring, access control, logging, backups and testing.

The initial programme funds one year of server capacity so the prototype can be built and evaluated in a production-like environment.

# Технології

VIRTUHAI планується як модульна програмна платформа, а не один монолітний застосунок. Її ключові рівні:

1. **Адаптери джерел** — контрольовані конектори до погоджених публічних API та наборів даних.
2. **Потік даних** — нормалізація, контекст часу й місця, походження даних і перевірки якості.
3. **Механізм обізнаності** — правила й дослідницькі моделі, що пріоритезують сигнали та пояснюють їх.
4. **Продуктовий рівень** — дашборд, сповіщення та майбутні інтерфейси для окремих продуктів.
5. **Операційний рівень** — захищений хостинг, моніторинг, контроль доступу, журнали, резервні копії та тестування.

Початкова програма фінансує серверні ресурси на один рік, щоб прототип можна було створити та оцінити в середовищі, наближеному до робочого.
